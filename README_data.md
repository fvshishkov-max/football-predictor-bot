# Структура данных проекта

## 📁 data/ - Основная папка данных

### 📁 predictions/
- `predictions.json` - история всех предсказаний
- Формат: JSON с массивами предсказаний

### 📁 history/
- `matches_history.db` - SQLite база данных истории матчей
- `predictions_history_*.csv` - CSV экспорты истории предсказаний
- `signals_history_*.csv` - CSV экспорты истории сигналов

### 📁 stats/
- `prediction_stats.json` - статистика предсказаний
- `predictions_stats_*.txt` - текстовые отчеты статистики

### 📁 logs/
- `app.log` - основной лог приложения
- Другие лог-файлы

### 📁 backups/
- Автоматические бэкапы важных файлов
- Файлы с расширениями `.backup`, `.back`

### 📁 models/
- `xgboost_model.pkl` - обученная XGBoost модель
- `xgboost_model_features.json` - признаки модели

### 📁 cache/
- `football_cache.db` - кэш API запросов
- Временные данные для ускорения работы

## 📁 archives/ - Архивы старых данных

- `backup_YYYYMMDD_HHMMSS.zip` - архивы старых файлов
- Создаются автоматически при очистке

## 🔄 Автоматическая очистка

Запустите `full_cleanup.bat` для:
1. Создания структуры папок
2. Организации файлов
3. Архивирования старых данных
4. Очистки временных файлов
