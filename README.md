# ⚽ AI Football Predictor Bot

Бот для анализа футбольных матчей в реальном времени и отправки сигналов о вероятности голов в Telegram.

## 📊 Возможности

- 🔴 **LIVE мониторинг** матчей через SStats API
- 📈 **Анализ статистики** с самообучением (XGBoost)
- ⚽ **Прогнозирование голов** с учетом xG (Expected Goals)
- 🤖 **Автоматическая отправка** сигналов в Telegram
- 📊 **Детальная статистика** по прогнозам
- 🌍 **Поддержка перевода** стран и лиг с флагами
- 🧠 **Умный анализ** MatchAnalyzer для отбора матчей
