# Football Predictor Bot

## Описание
Бот для анализа футбольных матчей в реальном времени и отправки сигналов о вероятности голов в Telegram.

## Текущая версия: 2.0 (11.03.2026)

## Структура проекта
- `app.py` - главный файл приложения
- `predictor.py` - логика прогнозирования голов
- `telegram_bot.py` - отправка сообщений в Telegram
- `api_client.py` - клиент для SStats API
- `models.py` - модели данных
- `ui.py` - графический интерфейс
- `config.py` - конфигурация
- `bot_state.py` - управление состоянием бота
- `database.py` - работа с базой данных
- `run.py` - стандартный запуск
- `run_fixed.py` - исправленный запуск

## Файлы данных
- `bot_stats.json` - статистика отправленных сигналов
- `signal_accuracy.json` - точность прогнозов
- `signals_history_latest.json` - последние 50 сигналов
- `football_bot.log` - логи работы
