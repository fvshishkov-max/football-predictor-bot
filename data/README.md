# Папка для данных бота

Эта папка содержит файлы данных, создаваемые ботом:

- `matches_history.db` - база данных истории матчей
- `predictions.json` - история предсказаний
- `xgboost_model.pkl` - обученная XGBoost модель
- `*.log` - файлы логов

**Важно**: Эти файлы не должны попадать в Git репозиторий.
